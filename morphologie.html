Codes morphologiques

(English version below)



Les caractères 3, 4, 6 et 7 sont utilisés pour indicer les unités morphémiques (les autres chiffres correspondent à des codes « machine » pour des phonèmes, voir « codes phonétiques »). La liste complète des unités morphologiques considérées est fournie dans le tableau en bas de page.

Flexions nominales de genre et de nombre: Le code 3 indice les flexions de genre et de nombre. Par exemple, le mot « photos » est codé en finale par le morphogramme <3s> auquel correspond le code phonologique <3#> (le caractère <#> est utilisé pour coder un graphème silencieux, voir « code phonétiques »). Par contre, dans le mot « parfois », le <s> final ne correspond pas à une flexion de nombre et le graphème final est donc simplement codé <s> auquel correspond le caractère <#>.

Flexions verbales : Le code 4 indice les flexions verbales. Par exemple, le mot « aiderai » est codé en finale avec <4ai> auquel correspond le code phonologique <4E>. Dans le mot <vrai>, le graphème <ai> ne correspond pas à une flexion verbale et est donc codé simplement <ai> auquel correspond <E>.

Finales de mots entendues dans les formes fléchies/dérivées (supports de flexion et/ou dérivation) : Le code 6 indice les différentes marques flexionnelles ou dérivationnelles de fin de mot. Par exemple, le <s> final de « gris » est codé <6s> auquel correspond le code <6#>. Il en est de même pour le <d> de « grand » (codé <6d>) qui sert d'appui à une forme fléchie (grande) et à une dérivation (grandeur).

Adverbes en <(m)ent> : Le code 7 indice le <ent> des adverbes de manière, en « ment ». Par exemple, <7ent> est associé au code phonologique <7@> dans le mot « simplement ».







The numbers 3, 4, 6 and 7 tag morphemic units in the database (remaining numbers are “machine code” tagging phonemes, see “Codes phonétiques”). The complete list of morphemic units tagged in the database is presented below:

Nominal inflections for gender and number are tagged with the number 3. For example, the grapheme “s” marking plural (nominal inflection for number) in the word “photos” is coded <3s>. Its phonetic code is <3#>, as this grapheme is silent (see “codes phonétiques” section). However, the grapheme “s” in the word “parfois” is not an inflection at all, therefore it is coded <s> and its phonetic code is simply <#>.

Verbal inflections are tagged with the number 4. For example, the grapheme “ai” marks the verbal inflection in “aiderai”. It is coded <4ai>, and its corresponding phonetic code is <4E>. In the word «vrai», “ai” is not an inflection: this grapheme is coded <ai> and the corresponding phoneme <E>.

Silent word-ending that are no longer silent when the word is inflected or derivated: They are tagged with the number 6. For example, the final “s” in “gris” is coded <6s> and its corresponding phonetic code is <6#>. The same goes for the final “d” in «grand», with the corresponding grapheme and phoneme respectively coded <6d> and <6#>. The final grapheme is a support both for inflection (“gris” vs. “grise”/“grand” vs “grande”) and derivation (“gris” vs. “grisant”/“grand” vs. “grandeur”).

Adverbs ending in «(m)ent» are tagged by the number 7. Manners adverbials or adverbs ending in “(m)ent” mark the manner or the way something is accomplished in the sentence. For example, the grapheme “ent” in the word “simplement” is coded “7ent” and its corresponding phonetic code is <7@>.
