Advanced search: In the column on the left, user can assign the order each correspondance is to be introduced in the program (starting with 1 for the first grapheme of the progression). If several correspondances (or graphemes, or phonemes) are to be taught simultaneously, enter the same value. If several associations have already been taught previously specify 1 next to each one them, and start with 2 for the new associations to teach. If a progression is based on graphemes or phonemes, all associations including that specific grapheme or phoneme are included.

The list of associations is organized according to phonemes, oral and nasal vowels, semi-vowels, occlusive consonants, fricatives, nasals and others. The associations concerning monophonemic verbal inflections (morphological index: 4) are grouped at the end of the list.
